# Scenic 팔로업 메일 템플릿

> 빈칸(`{ }`)을 채워 사용. 섹션은 2~3개 권장. 마지막 섹션은 항상 Next Step.

---

**제목:** `[Scenic] {미팅 성격} 팔로업`

**본문:**

```
{이름}님 안녕하세요,

{시간 내준 것에 대한 감사 + 상대 작품/미팅에 대한 개인화된 따뜻한 코멘트 한 문장}

{미팅 내용 정리하여 메일 드린다는 전환 문장 — 예: "미팅 내용과 다음 스텝 정리하여 메일 드립니다."}

*1/ {섹션1 제목}*

   - {핵심 내용}
   - {핵심 내용}
   - {핵심 내용}

*2/ {섹션2 제목}*

   - {핵심 내용}
   - {핵심 내용}

*{N}. Next Step*

{상대방에게 필요한 액션을 명확히. 필요한 항목은 불릿으로:}
   - {요청 항목}
   - {요청 항목}

{데드라인 명시 + 후속 미팅 제안 — 예: "가능하시다면 다음 주 수요일까지 전달 부탁드리며, 검토 후 후속 미팅을 진행하면 좋을 것 같습니다."}

{(선택) 기대감 표현 한 문장}

감사합니다,
{$FOLLOWUP_FROM_NAME} 드림
```

---

## 채워진 예시 (참고용)

```
이나님 안녕하세요,

오늘 이른 시간부터 귀한 시간 내어 미팅해 주셔서 감사합니다. 말씀 나누며 이나님의 작품 세계와 제작 방식에 대해 더욱 깊이 이해할 수 있는 좋은 시간이었습니다.

금일 논의 내용을 정리하여 공유드립니다.

*1/ 오리지널 시리즈 공동 제작 제안*

   - 자체 제작 툴 및 무검열 기업용 시댄스 api 지원
   - 3D 배경 애셋 제작 및 활용 지원
   - 시나리오 보조 작가 지원
   - MG(Minimum Guarantee) 제공

*2. Next Step*

작품 제안서 전달 부탁드립니다. 형식은 자유이며, 아래 내용이 포함되면 좋겠습니다.
   - 캐릭터 컨셉 / 시놉시스 / 세계관 설정

가능하시다면 다음 주 수요일까지 전달 부탁드리며, 검토 후 후속 미팅을 진행하면 좋을 것 같습니다.

함께 좋은 작품을 만들어갈 수 있기를 기대하겠습니다.

감사합니다,
{$FOLLOWUP_FROM_NAME} 드림
```
